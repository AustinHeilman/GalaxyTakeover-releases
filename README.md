# GalaxyTakeover

A space 4X strategy game inspired by Space Empires IV.

## Download

Go to the [Releases](https://github.com/AustinHeilman/GalaxyTakeover-releases/releases) page to download the latest version.

### Available Platforms

- **Windows:** Download the `-Windows-Setup.exe` installer. Requires [.NET 10 Desktop Runtime x64](https://dotnet.microsoft.com/download/dotnet/10.0) (installer will prompt if needed).
- **macOS (Apple Silicon):** Download the `-Apple-Silicon.zip` for M1/M2/M3/M4 Macs. Self-contained — just unzip and run.
- **macOS (Intel):** Download the `-Apple-Intel.zip` for older Intel Macs. Self-contained — just unzip and run.
